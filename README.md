# netty client

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
- generate dist/
- copy dist/ to nginx/html/
- run command `start nginx` to start the application
- run command `nginx.exe -s quit` to quit the application
